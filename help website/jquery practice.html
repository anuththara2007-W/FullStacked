Complete jQuery Guide - Simple & Easy

1. What is jQuery?

jQuery is a JavaScript library that makes coding easier and faster. Think of it like a toolbox that helps you do cool things on websites without writing lots of complicated code!

2. Getting Started with jQuery

How to Add jQuery to Your Page

<!-- Add this line in your HTML <head> section -->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>

üåü Complete Example Structure

Here is how the <body>, elements, and script work together: 

</head>
<body>

<p id="para">This is a paragraph</p>

<button id="hideBtn">Hide button</button>
<button id="showBtn">Show button</button>

<script>
$(document).ready(function(){

  $("#hideBtn").click(function(){
    $("#para").hide();
    alert("Paragraph Hidden");

  });

</script>

Basic Syntax

$(selector).action()
  • $ = jQuery magic symbol
  • selector = finds HTML elements
  • action() = does something to those elements

3. jQuery Selectors (Finding Elements)

1. Basic Selectors

Element Selector - Finds elements by tag name 

$("p")        // Finds all paragraphs
$("div")      // Finds all divs
$("button")    // Finds all buttons

ID Selector - Finds element by ID (use #)

$("#myId")     // Finds element with id="myId"

Class Selector - Finds elements by class (use .) 

$(".myClass")  // Finds all elements with class="myClass"

2. Position Selectors

$("p:first")      // First paragraph
$("p:last")       // Last paragraph
$("li:even")      // Even numbered list items (0, 2, 4...)
$("li:odd")       // Odd numbered list items (1, 3, 5...)
$("p:eq(2)")      // Third paragraph (counting starts at 0)

4. jQuery Effects (Making Things Move!)

1. Hide and Show

$("#box").hide();        // Makes element disappear instantly
$("#box").show();        // Makes element appear instantly
$("#box").toggle();      // If visible, hide it. If hidden, show it!

// With Speed
$("#box").hide("slow");     // Disappears slowly
$("#box").show("fast");     // Appears quickly
$("#box").hide(1000);       // Takes 1000 milliseconds (1 second)
