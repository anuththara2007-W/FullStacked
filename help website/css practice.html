              üìù 20 Rapid Questions
            

            

              Click to reveal (Answers with Code)
            


            

              

                1. What does the acronym CSS officially stand for in web
                development?‚ñº
              

              

                Cascading Style Sheets used for styling.
                style.css
              

            

            

              

                2. How do you correctly select an element by its unique ID in
                CSS?‚ñº
              

              

                Use the hash symbol followed by ID. #header {}
              

            

            

              

                3. How do you select elements by their class name in a
                stylesheet?‚ñº
              

              

                Use a dot before the class name. .btn {}
              

            

            

              

                4. What behavior does the display: flex property apply to a
                container?‚ñº
              

              

                Enables flexible layout for child elements.
                display: flex;
              

            

            

              

                5. What does the justify-content property control in a flex
                container?‚ñº
              

              

                Aligns items along the main axis.
                justify-content: center;
              

            

            

              

                6. What does the align-items property control in a flex
                container?‚ñº
              

              

                Aligns items along the cross axis.
                align-items: center;
              

            

            

              

                7. What are the four distinct parts that make up the Box
                Model?‚ñº
              

              

                Margin, border, padding, and content.
                box-sizing: border-box;
              

            

            

              

                8. How does position: absolute place an element within the
                document?‚ñº
              

              

                Relative to nearest positioned ancestor.
                position: absolute; top: 0;
              

            

            

              

                9. How does position: fixed place an element on the user's
                screen?‚ñº
              

              

                Positioned relative to the viewport.
                position: fixed; bottom: 0;
              

            

            

              

                10. What does the z-index property control regarding overlapping
                elements?‚ñº
              

              

                Stack order of positioned HTML elements.
                z-index: 999;
              

            

            

              

                11. What elements does the asterisk (*) selector target in
                CSS?‚ñº
              

              

                Selects every single element on page.
                * { margin: 0; }
              

            

            

              

                12. What is the reference base for the 'rem' CSS unit size?‚ñº
              

              

                Font size of the root element. font-size: 1.5rem;
              

            

            

              

                13. What does the 'vh' unit represent in CSS sizing terms?‚ñº
              

              

                One percent of the viewport height. height: 100vh;
              

            

            

              

                14. What happens to the layout space when visibility is
                hidden?‚ñº
              

              

                Hidden but preserves the element space.
                visibility: hidden;
              

            

            

              

                15. What happens to the layout space when display is set to
                none?‚ñº
              

              

                Removes element completely from layout.
                display: none;
              

            

            

              

                16. What is the effect of adding !important to a style
                rule?‚ñº
              

              

                Overrides all other distinct style rules.
                color: red !important;
              

            

            

              

                17. What does the transition property do to changing CSS
                values?‚ñº
              

              

                Smoothly changes values over time.
                transition: all 0.3s;
              

            

            

              

                18. Can you give a common example of a CSS pseudo-class?‚ñº
              

              

                Hover state is a pseudo class.
                a:hover { color: red; }
              

            

            

              

                19. What is the main difference between CSS Grid and CSS
                Flexbox?‚ñº
              

              

                Grid is 2D, Flexbox is 1D.
                display: grid;
              

            

            

              

                20. What is the primary purpose of using Media Queries in
                CSS?‚ñº
              

              

                Styles based on device characteristics.
                @media (max-width: 768px) {}
